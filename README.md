# Project Title

Check current cryptocurrency rates with <b>Coin</b>Co easily.

## Description

Using this application, you may check current rates for the most popular cryptocurrencies:
* <b>BTC</b> - Bitcoin
* <b>DOGE</b> - Dogecoin
* <b>ETH</b> - Ethereum
* <b>LTC</b> - Litecoin
* <b>XMR</b> - Monero
* <b>XRP</b> - Ripple
* <b>XVG</b> - Verge
* <b>ZEC</b> - Zcash
